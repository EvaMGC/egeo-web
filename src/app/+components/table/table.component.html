Because table is able to render any content in its cells, you are responsible for adding content to it from your view component. Next, we explain you how to build your own table.

First of all, you have to call to the table component introducing the list of header names. Moreover, you can specify if header is visible or not and if rows can be sortable.

Once table structure is defined, you have to add rows to your table. Rows have to be created using tr tag according to the HTML table specifications. Moreover, st-table-row attribute is needed in order to call to our row component.

Each created row will be a cell container. Cells are created using td tag and st-table-cell attribute. In order to row knows what cells it has to render, you have to add st-table-row-content to all its children cell. Moreover, you can specify if row is displayed compacted or not. Next you can see how a cell has to be defined.

Now that you know how to build each piece of our table, you can create your table adding as many rows and cells as you need. Next, you can see an example of a simple table with user data.
